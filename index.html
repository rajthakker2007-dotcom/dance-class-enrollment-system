About the Academy

What is this system?

The Dance Class & Choreography Enrollment System is a mini project that allows students to register for dance classes, choose their preferred style and level, select batch timing and also request choreography for events like weddings and college fests.

Key Features / Modules

  • Home Page & About Module
  • Dance Styles Display Module
  • Student Registration Module
  • Dance Style, Level & Batch Selection Modules
  • Choreography Type Selection Module
  • Live Enrollment Preview & Fee Estimation Module
  • Gallery & Testimonials Module
